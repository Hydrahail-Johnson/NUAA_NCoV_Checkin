# i·南航每日自动打卡
## 🍕项目说明

基于[Qiandao](https://github.com/AragonSnow/qiandao)框架，每日自动上报健康信息，支持校内打卡及校外打卡。

- 定时打卡
- 随机时间区间打卡
- 打卡结果微信通知

声明：本项目仅供学习研究使用，请遵守相关规定，出现任何后果概不负责。
